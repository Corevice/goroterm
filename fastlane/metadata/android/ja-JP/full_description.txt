GoroTerm は、スマートフォン / タブレットで本格的なターミナル体験を提供する、モバイルファーストの SSH クライアントです。

主な機能

• パスワードと鍵認証に対応したフル機能の SSH クライアント（パスフレーズ付き暗号化鍵にも対応）
• 初回接続時 / 鍵変更時のホストキー検証で中間者攻撃から保護
• xterm ベースのターミナル — マウス・タッチ・IME・ANSI カラー対応
• マルチタブ対応（ドラッグで並べ替え可能）
• tmux セッションマネージャ — 一覧・接続・作成・名前変更・終了、複数セッションの一括展開
• SFTP ファイルブラウザ — アップロード・ダウンロード・プレビュー・名前変更・削除
• サーバーリソースモニター — CPU 負荷・メモリ・ディスク・上位プロセス
• Claude Code 使用量トラッカー（5 時間 / 7 日制限・プラン情報）
• 音声入力（100+ 言語の長文ディクテーション対応）
• 画像・クリップボードのペースト（ffmpeg による GIF 変換対応）
• フォアグラウンドサービスでバックグラウンド時も SSH 接続を維持
• Play Store からのアプリ内アップデート通知
• 省電力モード — keepalive 間隔をユーザーが調整可能

プライバシー第一

• すべてのデータは端末内のみに保存
• 認証情報は Android Keystore で保護
• 解析・テレメトリ・第三者へのデータ共有は一切なし

多言語対応

英語・日本語・インドネシア語に対応。端末の言語設定に自動追従します。

開発者・システム管理者・外出先で安定したシェルアクセスが必要なすべての方のために設計されています。
