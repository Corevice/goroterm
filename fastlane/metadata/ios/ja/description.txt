Goroterm はリモートサーバへのアクセスを iPhone / iPad から快適に行うための SSH ターミナルクライアントです。

主な機能

• マルチセッション タブ — 複数サーバへの同時接続、同一ホストへの複数シェル切替を瞬時に
• tmux 統合 — セッション一覧・アタッチ・新規作成・リネーム・削除を UI から直接操作。再接続時にシームレスに復帰
• SFTP ファイルブラウザ — リモートディレクトリ閲覧、ファイルプレビュー、バックグラウンド転送対応のアップロード・ダウンロード
• 音声入力 — オンデバイスの音声認識でコマンドを発話入力
• クリップボード画像ペースト — 対応サーバ上のターミナルへ画像を送信
• クイックアクションバー — Ctrl・Esc・Tab・矢印・PgUp/PgDn・Ctrl+J 等を常時 1 タップでアクセス
• サーバ監視 — 接続中ホストの CPU・メモリ・ディスク使用率を表示
• Claude 利用ダッシュボード — サーバで Claude CLI を使用している場合のトークン消費量表示
• 安全な認証情報保管 — パスワード・秘密鍵は端末内に暗号化保存。クラウド同期一切なし
• ホスト鍵検証 — known_hosts ワークフローによる変更検出
• 自動再接続 — ネットワーク切替・バックグラウンド遷移・一時的な切断に自動対応
• ダークテーマ標準 — 長時間の作業でも目に優しい配色

モバイルでも快適に使えるよう設計: スムーズなスクロール、適切な IME 処理、サブメニューを掘らずに済むクイックアクションバー。バックグラウンドでも接続が維持され、戻ったら作業を続行できます。

プライバシー

SSH 認証情報は端末から外に送信されません。Goroterm は分析データ・トラッキング識別子・利用ログを一切収集しません。詳細はプライバシーポリシーをご確認ください。

ご利用には接続先の SSH サーバが必要です。
